# SEO Keyword Strategy — KDP X-Ray Helper

## Market Analysis

**Competitive landscape**: As of early 2026, there is NO other Chrome extension that automates KDP X-Ray entity editing. Existing KDP extensions (KDP Uploader, Productor for KDP, Auto-Fill) focus on book uploading/metadata — none touch X-Ray.

**Author pain points** (from KDP Community, Goodreads, indie author blogs):
- X-Ray editing is **tedious** — must click each entity one by one
- No bulk/batch editing in Amazon's native tool
- Data loss when manuscript is updated (overwrites custom descriptions)
- Authors copy-paste from glossaries to save time
- Many authors skip X-Ray entirely because it's too time-consuming
- 200+ entities for a novel = hours of manual work

**Key insight**: Authors don't search for "X-Ray Chrome extension" because they don't know such a tool could exist. They search for **the problem**, not the solution.
